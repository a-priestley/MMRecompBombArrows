# Majora's Mask: Recompiled - Bomb Arrows 🏹+💣

Adds the ability to combine arrows with bombs like in other Zelda titles.
Fasten a bomb to the tip of an arrow, causing it to detonate on impact,
or when the fuse expires.

![Demonstration](https://github.com/a-priestley/MMRecompBombArrows/blob/main/readme/demo.gif?raw=true)

## Usage

To use bomb arrows, assign both the Hero's Bow and Bombs to the same control button.
Each bomb arrow consumes one arrow and one bomb.

## Dependencies

[Interface Helper](https://github.com/t0mtee/MMRecompInterfaceHelper/), an excellent library
facilitating the custom Bomb Arrows equipment icon
(just a scaled-down bomb icon composited with the Hero's Bow icon).
